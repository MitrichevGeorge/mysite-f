# GitHub Markdown Демонстрация
## Основные возможности

### 1. Заголовки
# H1 - Заголовок первого уровня
## H2 - Заголовок второго уровня
### H3 - Заголовок третьего уровня
#### H4 - Заголовок четвертого уровня
##### H5 - Заголовок пятого уровня
###### H6 - Заголовок шестого уровня

### 2. Форматирование текста
**Жирный текст** с помощью двойных звездок
*Курсивный текст* с помощью одинарных звездок
~~Зачеркнутый текст~~ с помощью тильд
__Жирный курсив__ с помощью комбинации

### 3. Списки
#### Маркированные списки
- Пункт 1
- Пункт 2
  * Подпункт 1
  * Подпункт 2
- Пункт 3

#### Нумерованные списки
1. Первый пункт
2. Второй пункт
3. Третий пункт

### 4. Блоки кода
```python
def example_function():
    print("Это пример кода")
    return True
```

### 5. Ссылки
[Ссылка на GitHub](https://github.com)
[Ссылка на документацию](https://docs.github.com "Тултип")

### 8. Таблицы

### 9. Диаграммы
```mermaid
flowchart LR
    A[Начало] --> B{Условие?}
    B -->|Да| C[Действие 1]
    B -->|Нет| D[Действие 2]
    C --> E[Конец]
    D --> E
```
