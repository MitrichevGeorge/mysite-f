## Само условие задачи:

Вам вводится 2 строчки: строка содержащая пробелы и несколько чисел через пробелы. Надо вывести куски превой строки в соответствии с порядком индексов написаннных во второй

### Пример:

на ввод

a b c d

1 3 1 0 2

Ваш код должен выдать:

bdbac
